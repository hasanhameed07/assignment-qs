# assignment-qureos

The assignment is done on NodeJs using Typescript.
libraries used:
- express
- tsc-watch
- axios
- dotenv
- helmet
- http-status-codes
- moment

## How to run locally
    npm install
    npm run dev

## Task 1
  Get Index of Characters - "GET /api/indexes/Sample"
  
## Task 2
  Get Movies Sorted - "GET /api/movies/Spiderman"
  
## Task 3
  Get Stocks - "Get Stocks - "GET /api/stocks/5-January-2000"
